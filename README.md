# Users Subscriptions API

- Добавить событие о подписке пользователя
- Получение аналитики по интервалу времени(средний период времени оформленных подписок)

## Используемый стек технологий

- Java 16
- Spring Boot 2.6
- Swagger UI

- Helm
- JDBC
